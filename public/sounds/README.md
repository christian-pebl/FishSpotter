# Sound effects

Add these short MP3 files for in-game feedback (use royalty-free sources e.g. Pixabay, Freesound):

- `correct.mp3` – plays alongside the confetti explosion when the user's answer is correct (drop your custom celebration sound here)
- `wrong.mp3` – play when the answer is wrong
- `streak.mp3` – play when the user extends their streak

If files are missing, the app will not break; playback is skipped.
